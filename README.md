# Women's Day Special Website 🌸

This project is a special website created for Women's Day to celebrate and appreciate women.

## Features
- Beautiful animations 🌸
- Images for Women's Day 🖼️
- Music and celebration effects 🎶

This website is created to show respect, love, and appreciation to all women on International Women's Day.

## Technologies Used

- CSS
- JavaScript

Open the website using the GitHub Pages link or scan the QR code to view the celebration page.
